At this time, Python support for Azure functions is at experimental stage. This note captures the steps for installing packages needed. First go to CMD prompt in Kudu console. - function directory: cd D:\home\site\wwwroot - create a new virtualenv: "D:\Python27\Scripts\virtualenv.exe azenv" - activate it: "azenv/Scripts/activate.bat" - upgrade pip: "pip install --upgrade pip" - download python wheels for windows from Christoph Gohlke's repository: http://www.lfd.uci.edu/~gohlke/pythonlibs - using Azure console, upload to function - pip install .\.whl
