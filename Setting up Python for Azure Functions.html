At this time, Python support for Azure functions is at experimental stage. This note captures the steps for installing packages needed. First go to CMD prompt in Kudu console. - function directory: cd D:\home\site\wwwroot - create a new virtualenv: "D:\Python27\Scripts\virtualenv.exe azenv" - activate it: "azenv/Scripts/activate.bat" - upgrade pip: "pip install --upgrade pip" - download python wheels for windows from Christoph Gohlke's repository: http://www.lfd.uci.edu/~gohlke/pythonlibs - using Azure console, upload to function - pip install .\.whl And in the python function import as follows import sys sys.path.append(os.path.abspath(os.path.join(os.path.dirname( __file__ ), '..', 'lib'))) sys.path.append(os.path.abspath(os.path.join(os.path.dirname( __file__ ), '..', 'azenv/Lib/site-packages'))) # lib folder is where one could upload other python scripts as import modules.
